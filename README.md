## 📌 Descrição

Este repositório reúne as **atividades, exercícios e projetos desenvolvidos na disciplina de Sistemas Web I.** Ao longo da matéria, foram trabalhados conceitos fundamentais para a construção de aplicações web, desde a estrutura de páginas até a interação com o usuário e processamento de dados.

## 🎯 Objetivo

**Desenvolver conhecimentos básicos e intermediários no desenvolvimento web,** permitindo a criação de páginas e sistemas simples, utilizando boas práticas de organização, lógica de programação e integração entre front-end e back-end.
